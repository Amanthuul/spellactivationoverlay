# SpellActivationOverlay 2.3.2
@here I am pleased to announce the newest release of SpellActivationOverlay, version 2.3.2

## Classes
### :hunter:  Hunter
New Spell Alert:
- Mongoose Bite\* (Era, TBC)

\* The effect was already supported as glowing button. This update adds a spell alert on top of it.
### :priest:  Priest
Mind Spike has been removed temporarily until it is resolved (Season of Discovery)
### :shaman:  Shaman
Updates Glowing Button:
- Molten Blast no longer glows infinitely after leaving combat (Season of Discovery)

## Bug Fixes
### Options
The options panel would not work with the upcoming Firelands patch (Cataclysm). This has been fixed for Cataclysm PTR, and this fix is compatible with Live.
